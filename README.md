# 🤖 Chatbot Inteligente  

Este é um **chatbot responsivo**, moderno e interativo, capaz de responder perguntas diretamente com pesquisas no Google. Além disso, possui um **design colorido e dinâmico**, com um botão para limpar o chat e um fundo animado que muda de cor automaticamente! 🎨✨  

## 🚀 Funcionalidades  

✅ **Responde perguntas automaticamente**  
✅ **Pesquisa no Google integrada**  
✅ **Interface moderna e responsiva** (funciona em PC, tablet e celular)  
✅ **Botão de limpeza do chat**  
✅ **Efeito de mudança de cor automático**  

## 📦 Instalação  

1. Clone o repositório:  
   ```
   git clone https://github.com/seu-usuario/chatbot-inteligente.git
   ```
2. Navegue até a pasta do projeto:  
   ```
   cd chatbot-inteligente
   ```
3. Abra o arquivo `index.html` no navegador.  

## 🎯 Como Usar  

1. Digite sua pergunta na caixa de texto.  
2. Clique em "Enviar" ou pressione **Enter**.  
3. O chatbot fornecerá um link para pesquisa no Google com a sua pergunta.  
4. Se quiser limpar o chat, clique em "**Limpar Chat**".  

## 🎨 Personalização  

📌 Para mudar as cores do fundo, edite o arquivo **style.css**, modificando os valores da animação `@keyframes changeBackground`.  
📌 Para adicionar respostas customizadas, edite `getBotResponse()` no **script.js**.
